# Retail Sales Executive Summary Dashboard (Power BI)

This project features an interactive <strong>Executive Summary Dashboard</strong> developed using <strong>Microsoft Power BI</strong>. Designed for retail analytics, this dashboard provides stakeholders with a high-level overview of business performance, enabling deep-dive analysis through interactive filtering.

## 📊 Dashboard Overview
The dashboard centralizes disparate retail data into a single, cohesive view, tracking key business health metrics and performance trends across regions, categories, and store types.

## 📈 Visualizations & Insights

* <strong>Net Sales by Month:</strong> A line chart visualizing seasonal sales trends and identifying peak performance periods.
* <strong>Net Sales by Region:</strong> A bar chart providing a geographical breakdown of revenue distribution.
* <strong>Total Profit by Category:</strong> Analyzes the profitability of different product segments (e.g., Electronic Accessories vs. Fashion).
* <strong>Store Type Breakdown:</strong> A donut chart comparing performance across Supermarkets, Mini Stores, and Hypermarkets.
* <strong>Top 5 Products:</strong> Ranks top-selling products to highlight market demand.
* <strong>Payment Mode Analysis:</strong> Visualizes customer preference between UPI, Cards, Cash, and Wallets.

## 🎛️ Interactive Filters (Slicers)
The dashboard includes dynamic slicers for:
* <strong>Year & Month:</strong> For time-series analysis.
* <strong>Category:</strong> To isolate specific product performance.
* <strong>Region:</strong> To drill down into local market dynamics.

## 🛠️ Tech Stack
* <strong>Tool:</strong> Microsoft Power BI Desktop.
* <strong>Data Processing:</strong> Power Query (ETL).
* <strong>Visualization:</strong> Interactive charts, DAX (Data Analysis Expressions), and Slicers.
